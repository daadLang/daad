# Changelog

All notable changes to Daad will be documented in this file.

## [Unreleased]

### Added
- Python-style `import` and `from ... import ...` parsing.

### Changed
- Parser and AST work continue to support Python-style import syntax.


## [v0.1.0] - 2026-05-06

### Added
- Basic object-oriented programming support.
- Class definitions, attribute assignments, method calls, and single inheritance.
- OOP internals documentation and related examples.

### Changed
- Interpreter and AST expanded to support class-based execution.
- Example class instantiation and inheritance behavior were added.

## [v0.0.0] - 2026-05-06

### Added
- Initial Daad language runtime foundation.
- Lexer with indentation-aware tokenization.
- Recursive-descent parser and AST structure.
- Core interpreter with variables, expressions, functions, collections, and control flow.
- Built-in operations and initial standard behavior.
- Project documentation, examples, and test suite.
